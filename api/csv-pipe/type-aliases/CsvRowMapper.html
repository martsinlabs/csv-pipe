Type Alias: CsvRowMapper<T> ​

ts
type CsvRowMapper<T> = (record, context) => T;

Defined in: types.ts:135

Map each raw parsed record into the final element. Use it to validate (for example with a schema library) or reshape a record. The returned value is what parse yields, so this is where runtime type safety can be enforced.
